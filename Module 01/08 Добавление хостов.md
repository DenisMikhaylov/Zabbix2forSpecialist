### **Задача 1: Настройка подключения агента **

1. открыть веб инетерфейс Zabbix

2. Добавление без агентового узла 

```
Host name: google.com
  interface agent: DNS  google.com
  New group: my ext
  Tag:
    host goggle
    my: my
```
